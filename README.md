### install project
```
//clone this project
git clone git@github.com:wudtichaikarun/practice-sass-flex_box.git

// install node_modules
npm install
```

### develop mode
```
// edit update delete at sass/*
npm run start
```

### compile,concat,prefix,compress scss to one file -> css/main.css
```
// build sass
npm run build
```
